# TikTok Downloader

A Python-based TikTok video downloader with support for downloading videos by username or trending videos.

## Features

- Download videos from any public TikTok user
- Download trending videos
- Sort by most recent, most viewed, or oldest
- Progress tracking with real-time status
- Smart file naming preserves sort order

## Installation

```bash
pip install -r requirements.txt
python -m playwright install
```

## Usage

```bash
python main.py
```

## Credits

Uses [TikTokApi](https://github.com/davidteather/TikTok-Api) for metadata and [SnapTik](https://snaptik.app) for downloads.
